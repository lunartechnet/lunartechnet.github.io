Our Mission

At Lunartech, we aim to make outdated operating systems easier to use for industry in the modern day. This is mainly achieved through the use of small, easy-to-use commandline programs meant to integrate seamlessly into any operating system.

 

Software

All Lunartech software for a specified platform will come pre-packaged with any and all binary softwares required in a zipped file. All binaries in the zipped file will install themselves into the commandline interface for use in the operating system.
